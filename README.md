# # 🗳️ MediCoreApi

API REST desarrollada en ASP.NET Core 8 para la gestión de un sistema de citas medicas. Proyecto con buenas prácticas, arquitectura limpia.

---

## 🛠️ Tecnologías

- **ASP.NET Core 8** — Framework principal
- **Entity Framework Core** — ORM para acceso a datos
- **SQL Server** — Base de datos relacional
- **Swagger / Swashbuckle** — Documentación interactiva de endpoints
